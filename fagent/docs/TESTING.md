# Тестовые запросы для fagent (с пустой памятью)

## 1. Базовая проверка

### Тест 1.1: Простой диалог
```
Привет! Как дела?
```
**Ожидаемый результат**: Агент отвечает приветствием

### Тест 1.2: Проверка памяти о себе
```
Расскажи о себе. Что ты умеешь?
```
**Ожидаемый результат**: Агент описывает свои возможности (должен использовать скил fagent)

---

## 2. Проверка инструментов

### Тест 2.1: Файловая система
```
Создай файл test.txt с текстом "Hello World"
```
**Ожидаемый результат**: Файл создан, агент подтверждает

### Тест 2.2: Чтение файла
```
Прочитай файл test.txt
```
**Ожидаемый результат**: Агент читает и показывает содержимое

### Тест 2.3: Shell команды

```
Выполни команду: echo "Test command"
```
**Ожидаемый результат**: Команда выполнена, вывод показан

## 3. Проверка памяти

### Тест 3.1: Запоминание информации
```
Запомни: мой любимый цвет - синий
```
**Ожидаемый результат**: Агент подтверждает запоминание

### Тест 3.2: Извлечение из памяти (следующий turn)
```
Какой мой любимый цвет?
```
**Ожидаемый результат**: Агент отвечает "синий" (из контекста сессии)

### Тест 3.3: Поиск в памяти
```
Используй memory_search для поиска информации о цвете
```
**Ожидаемый результат**: Агент находит информацию о синем цвете

### Тест 3.4: Получение сущности
```
Используй memory_get_entity для поиска сущности "любимый цвет"
```
**Ожидаемый результат**: Агент находит или создает сущность

## Быстрый тест (5 минут)

Для быстрой проверки выполни только эти тесты:

1. **Базовый диалог**: "Привет! Расскажи о себе"
2. **Файл**: "Создай файл test.txt с текстом Hello"
3. **Память**: "Запомни: мой любимый цвет синий"
4. **Поиск**: "Используй memory_search для поиска информации о цвете"
5. **Скил**: "Где находится конфигурационный файл fagent?"

Если все 5 тестов прошли успешно - базовая функциональность работает! ✅

## Проверка логов

После тестов проверь логи на наличие ошибок:

```bash
# Проверка графовой индексации
sqlite3 ~/.fagent/workspace/memory/registry.db "SELECT status, error FROM graph_jobs ORDER BY created_at DESC LIMIT 5;"

# Проверка узлов графа
sqlite3 ~/.fagent/workspace/memory/registry.db "SELECT COUNT(*) FROM graph_nodes;"

# Проверка embeddings (NEW)
sqlite3 ~/.fagent/workspace/memory/registry.db "SELECT COUNT(*) FROM graph_node_embeddings;"
```

**Ожидаемые результаты**:
- graph_jobs: status = "done", error = ""
- graph_nodes: COUNT > 0
- graph_node_embeddings: COUNT > 0 (если semantic search работает)
